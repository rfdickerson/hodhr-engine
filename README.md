# HodhrEngine

## Features
- [x] Vertex buffer objects
- [x] Wavefront (OBJ) Loading
- [x] Textures

# Compiling:

1. Install the dependencies:

```
sudo apt-get install libglm-dev libglew-dev libassimp-dev libdevil-dev libglew-dev libsdl2-dev
```

2. Create a build directory:

```
mkdir build
cd build
cmake ..
make
```
